react-fukiso is a simple UI library for react.


## Getting Started

```bash
npm i --save react-fukiso
```

```jsx
import React from 'react'
import Fukiso from 'react-fukiso'

const App = () => (
  <div>
    サイト管理者を不当な逮捕から守るために、「了解」を押して下さい
    <Fukiso />
  </div>
)

export default App
```
